📊 Customer Churn Analysis – Power BI
Codveda Internship | Level 1
------------------------------------------------------------------------------------------------------
📌 Project Overview

This project was completed as part of the Codveda Power BI Internship – Level 1.

The main objective was to apply basic Power BI skills including:

Importing data

Cleaning and transforming data

Creating calculated columns

Building an interactive dashboard

The project analyzes customer churn data to understand customer behavior and churn patterns.

------------------------------------------------------------------------------------------------------

📂 Dataset

churn_powerbi.csv

A single CSV file was used for the entire analysis.

------------------------------------------------------------------------------------------------------

🛠 Data Preparation

The following steps were performed:

Imported the CSV file into Power BI

Checked and corrected data types

Removed duplicate records

Checked and handled null values

Created a conditional column (Service Call Category)

Created a DAX calculated column (Total Minutes)

------------------------------------------------------------------------------------------------------

📊 Dashboard Components

The final dashboard includes:

📌 Total Customers KPI

📌 Churn Distribution Pie Chart

📌 Customer Service Calls vs Churn Bar Chart

📌 State Slicer for interactive filtering

------------------------------------------------------------------------------------------------------

🔎 Key Insight

Customers with a higher number of customer service calls show a greater likelihood of churn.
